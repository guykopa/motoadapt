motoadapt
=========

Adaptive engine for motor rehabilitation exercises.

Automatically adjusts exercise difficulty based on patient performance
using a sliding window algorithm over the last 3 responses.
No database. No framework. No frontend.
Pure PHP 8.2, strict TDD, strict SOLID.
